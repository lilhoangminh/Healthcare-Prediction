Thư mục này chứa các file CSV cho Disease Symptom Prediction:

1. dataset.csv - Dataset chính với triệu chứng và bệnh
2. symptom_Description.csv - Mô tả các triệu chứng  
3. symptom_precaution.csv - Biện pháp phòng ngừa cho từng bệnh

Vui lòng tải các file CSV từ URL đã cung cấp và đặt vào thư mục này.
